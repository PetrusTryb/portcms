# PortCMS v2.0

Simple CMS for portfolios - second version.

  - [ ] Code (with syntax highlighting)
  - [ ] Video
  - [ ] Map

## Installation
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open http://localhost:3000/
